# TUGAS PRAKTIKUM
# SURPRISE PERTEMUAN KE-4


## Studi Kasus

Buatlah analisis dua program C++ berikut:
1. `task1.cpp` untuk pengurutan data angka menggunakan Bubble Sort dan Selection Sort.
2. `task2.cpp` untuk pencarian kata pada kalimat menggunakan konsep Sequential Search.


## Ketentuan

1. Jalankan dan pahami kedua file program (`task1.cpp` dan `task2.cpp`).
2. Jelaskan fungsi/prosedur yang digunakan pada masing-masing program.
3. Jelaskan alur input, proses, dan output dari kedua program.
4. Sertakan flowchart dan hasil output program.
5. Program di-copy dalam tabel, bukan dalam bentuk screenshot.


## Penjelasan Singkat Program

### A. `task1.cpp` (Pengurutan)

- Program menyiapkan data angka dalam array.
- Data diurutkan dengan Bubble Sort.
- Data yang sama juga diurutkan dengan Selection Sort.
- Hasil kedua metode ditampilkan untuk dibandingkan.


### B. `task2.cpp` (Pencarian)

- Program menerima kata yang ingin dicari.
- Kalimat dipindai dari kiri ke kanan dan dipisah per kata.
- Setiap kata dibandingkan dengan kata input.
- Program menampilkan posisi kata jika ditemukan.
