# TUGAS PRAKTIKUM
# SURPRISE PERTEMUAN KE-4


## Studi Kasus

Buatlah program C++ menggunakan `struct` identitas mahasiswa dengan elemen berikut:
- Nama
- NIM
- Kelas


## Ketentuan

1. Data identitas diinput secara manual melalui keyboard.
2. Lakukan pengurutan data berdasarkan NIM menggunakan:
	- Bubble Sort
	- Selection Sort
3. Tampilkan data sebelum dan sesudah diurutkan.
4. Cari dan tampilkan angka terakhir dari NIM setiap mahasiswa.
5. Sertakan flowchart dan hasil output program.
6. Program di copy didalam tabel bukan di screenshot
7. Hasil di sreenshot
